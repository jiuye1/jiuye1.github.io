事业单位概况

发表于 2015-06-06   |   分类于 公共基础

事业单位四条件：

  1. 依法设立；
  2. 从事公益服务；
  3. 不以盈利为目的；
  4. 社会组织。

事业单位四特征：

  1. 属于非公共权力机构，社会职能是提供专业性服务；
  2. 属于知识密集型组织；
  3. 经费来源的国家化；
  4. 范围的广泛性。

事业单位的分类：

  1. 承担行政职能的事业单位；
  2. 从事生产经营活动的事业单位；
  3. 从事公益服务的事业单位。
  4. 第一个逐步划归行政机构或转为行政机构，第二个逐步转为企业，第三个继续存在并加强。今后，不再设立第一和第二个。

事业单位存在的社会基础：

  1. 满足公共需要；
  2. 弥补市场缺陷；
  3. 3.节制政府失灵；
  4. 防止契约失效；
  5. 弥补志愿不足。

人事管理的基本原则：

  1. 公开平等原则；
  2. 竞争择优原则；
  3. 尊重知识、尊重人才原则；
  4. 依法管理和精简原则。

事业单位岗位类别：

  1. 管理岗位；
  2. 专业技术岗位；
  3. 工勤技能岗位。

公开招聘的原则：

  1. 坚持德才兼备的用人标准，贯彻公开、平等、竞争、择优的原则。
  2. 坚持政府宏观管理与落实单位用人自主权相结合，统一规范、分类指导、分级管理。

事业单位考核制度：

  1. 原则：客观公正、民主公开、注重实绩。
  2. 内容:德、能、勤、绩、廉。
  3. 等次：优秀、合格、基本合格、不合格。
  4. 考核方法：领导与群众相结合、平时与定期相结合、定性与定量相结合、采取综合分析、量化测评等符合岗位特点的方法。

人事管理制度存在的问题：

  1. 政事一体：按照党政机关干部人事管理制度管理各类事业单位人才，没有充分体现出事业单位特点和工作人员的工作性质。强化了"官本位"意识，不利于调动事业单位工作人员的积极性，也不利于社会发展。
  2. 缺乏科学的分类：人事管理制度与使行政机关的区别不大，无法反映不同事业单位及事业单位内部管理、技术、工勤等岗位、人员的不同性质与特点。无法针对性的建立不同的管理和激励模式，不利于事业单位各类人才的成长与积极性的调动。
  3. 缺乏用人权和自主性。国家按行政计划与行政命令配置资源，导致用人与知识的脱节，也不利于人才资源的优化配置。缺乏用人权，一切按行政计划与行政命令形事，事业单位缺乏根据实际需要实施人力资源管理与开发的自主性和积极性，造成干多干少一个样、能进不能出的人事管理弊端。
  4. 管理机制缺乏活力：套用行政单位的管理方式管理工作人员，缺少完整而系统的人力资源发展规划和人才培养与管理制度，人事工作缺乏创新和活力，管理手段单一落后。
