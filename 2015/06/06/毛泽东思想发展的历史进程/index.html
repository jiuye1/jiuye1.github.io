毛泽东思想发展的历史进程

发表于 2015-06-06   |   分类于 毛概

毛泽东思想的萌芽

  1. 时间：中国共产党的创立时期和国共合作的国民革命时期，即从1921年7月中国共产党成立到1927年7月国民革命失败。
  2. 标志：
    • 以毛泽东为代表的中国共产党人，提出了彻底的反帝反封建的民主革命纲领。
    • 提出了无产阶级在民主革命中的领导权问题和农民同盟军问题。
    • 阐明了中国革命的对象、动力和前途等问题。
    • 明确指出无产阶级是中国革命的领导力量，初步形成了新民主主义革命的基本思想。
  3. 代表作：
    • 《中国社会各阶级的分析》
    • 《湖南农民运动考察报告》
  4. 会议：
    • 中共二大建立了反帝反封建纲领。
    • 中共三大确立了第一次国共合作统一战线策略。
    • 中共四大形成了工农联盟的思想。

毛泽东思想的形成

  1. 时间：土地革命战争的初期和中期，即1927年“八七”会议到1935年1月遵义会议。
  2. 标志：
    • 以毛泽东为代表的中国共产党人，敢于同党内盛行的马克思主义教条化、共产国际和苏联经验神圣化作斗争。
    • 提出红色政权理论，“工农武装割据”思想
    • 提出土地革命和农村革命根据地建设
    • 明确提出了党的思想路线问题，强调不唯书、不唯上、要靠中国同志了解中国情况、把马克思主义理论同中国实际情况结合起来，在群众斗争中创造新局面
    • 初步阐述了毛泽东思想的活的灵魂是实事求是、群众路线、独立自主。
  3. 代表作：
    • 《中国红色政权为什么能够存在》
    • 《井冈山的斗争》
    • 《星星之火，可以燎原》
    • 《反对本本主义》
  4. 会议：
    • 八七会议提出了土地革命和武装斗争的基本路线。
    • 遵义会议确立了以毛泽东为核心的党中央的正确领导。

毛泽东思想的成熟：

  1. 时间：土地革命后期到抗日战争时期。
  2. 标志：毛泽东思想得到系统总结和多方面展开，成果是：
    • 形成了新民主主义革命理论的完整体系，系统的论述了新民主主义革命的对象、动力、领导、前途、纲领等一系列重要问题。
    • 总结出了党领导新民主主义革命的三大法宝。
    • 实现了毛泽东哲学思想体系的构建。
    • 新民主主义革命理论的形成是毛泽东思想成熟的主要标志。
  3. 代表作：
    • 《中国革命和中国共产党》
    • 《新民主主义论》
    • 《<共产党人>发刊词》
  4. 会议：
    • 党的七大，确立毛泽东思想为中国共产党的指导思想。
    • 这是毛泽东思想发展史上的一个里程碑。
